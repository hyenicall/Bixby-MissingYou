# bixby missingyou
##### 캡슐 기능 소개: 유기동물 습득정보를 지역과 날짜, 품종에 따라 제공하는 캡슐입니다.

1. input concept (순서상관 없음)
    + Text KeyWord(종) -필수
     + Text Where(어디서) - 옵션
      + Text When(언제) -옵션
2. 대표발화
    - 대표 발화#1: 경기도에서 이번주안에 유기된 포메라니안 보여줘
     - 대표 발화#2: 서울 마포구에서 잃어버린 유기묘 찾아줘
     - 대표 발화#3: 작년 강남에서 잃어버렸던 믹스견 보여줘
